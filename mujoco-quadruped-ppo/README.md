# MuJoCo Playground + PPO 四足机器人 Locomotion

## 项目目标

基于 MuJoCo Playground 搭建四足机器人 locomotion 任务，使用 PPO 完成训练、评估和结果整理，形成可复现、可讲解的主项目 1。

## 项目背景

该项目用于理解连续控制任务中的状态空间、动作空间、奖励设计、训练稳定性和策略评估流程，并积累机器人强化学习项目实践材料。

## 当前状态

项目目录和文档模板已建立，环境配置、任务选择、训练代码及实验验证尚待完成。当前不包含训练结果或实验结论。

## 计划完成内容

- 确定 MuJoCo Playground 四足机器人任务与评估协议。
- 完成环境安装、运行和随机策略测试。
- 梳理 observation、action、reward、termination 等任务定义。
- 建立 PPO 训练与配置管理流程。
- 记录训练曲线、评估视频和失败案例。
- 整理实验报告、复现步骤和面试讲解材料。

## 关键文件说明

- `notebooks/`：环境探索和结果分析笔记本。
- `videos/`：后续保存策略评估视频。
- `figures/`：后续保存训练曲线和结构示意图。
- `notes/`：调试记录、环境说明和阶段笔记。
- `report.md`：实验报告模板。
- `interview_notes.md`：项目面试讲解模板。
