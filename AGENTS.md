# Repository instructions

- 项目运行环境为 WSL Ubuntu 24.04。
- Python 环境统一使用 Conda 环境 `embodied-ppo`。
- 从仓库根目录执行 Python 命令，优先使用：
  `conda run -n embodied-ppo python <script>`
- 修改前先阅读相关代码；复杂或跨文件修改前简要说明方案。
- 修改完成后运行相关测试或最小验证；若无法运行，说明原因。
- 除非用户明确要求，不修改数据集、模型权重、密钥或凭据文件。
- 不读取、展示或提交密钥及其他敏感信息。
- 未经明确许可，不执行 `git commit`、`push`、`merge`、`rebase`、`reset`、`clean` 或删除操作。
- 保留用户已有的未提交修改，不覆盖或回退无关改动。
- 默认使用中文回复。
